# Cartas Claras

Microsite estático para validar una oportunidad SEO y de monetización en España alrededor de cartas, bajas y reclamaciones frecuentes.

## Páginas actuales

- `/`
- `/baja-voluntaria.html`
- `/baja-gimnasio.html`
- `/desistimiento-online.html`
- `/reclamacion-operadora.html`
- `/devolucion-cargo-sepa.html`

## Qué incluye

- landing principal
- páginas separadas por intención de búsqueda
- formularios ligeros
- copia al portapapeles
- impresión a PDF
- `robots.txt`
- `sitemap.xml`
- `404.html`
- configuración básica para Netlify

## Despliegue rápido

### Netlify

1. Crear un nuevo sitio desde esta carpeta.
2. Publicar el directorio raíz del proyecto.
3. Sustituir `https://example.com` en:
   - `index.html`
   - `robots.txt`
   - `sitemap.xml`
4. Conectar Search Console y Analytics.

### Vercel

1. Importar el proyecto como sitio estático.
2. Publicar el directorio raíz.
3. Sustituir `https://example.com` por el dominio final.

### GitHub Pages

1. Subir esta carpeta a un repositorio.
2. Activar GitHub Pages desde la rama principal.
3. Usar la raíz o `/docs` como carpeta de publicación.
4. Actualizar URLs absolutas del dominio.

## Antes de publicar

- reemplazar `example.com`
- revisar el copy legal final
- añadir Analytics y Search Console
- añadir schema FAQ/HowTo por página
- crear las siguientes páginas long-tail
